# openfoodfacts-to-sql

This is a project to convert the jsonl export file from openfoodfacts into a mysql/sql database

## How to use it

First of all you need to download the JSONL exportfile from openfoodfacts website(https://world.openfoodfacts.org/data)
and after that paste it in the root of the project.
After that, execute the main.go file with command (go run main.go)
